## LMS / LXP engagement KPIs

If you run a learning platform, track platform health as a leading indicator:

| KPI | Description |
|---|---|

| Voluntary learning rate | % of learning initiated by the learner (not assigned) |
| Community engagement | Posts, reactions, comments per week |
| Mobile usage rate | % of content consumed on mobile |
